🚶 平移速度命令（控制“往哪搬”）

| 键   | 作用 | 改变的变量                       | 含义         |
| --- | -- | --------------------------- | ---------- |
| `w` | 向前 | `env.x_velocity_poi += 0.1` | 载体期望前进速度 + |
| `s` | 向后 | `env.x_velocity_poi -= 0.1` | 载体期望前进速度 − |
| `a` | 向左 | `env.y_velocity_poi += 0.1` | 载体期望侧移速度 + |
| `d` | 向右 | `env.y_velocity_poi -= 0.1` | 载体期望侧移速度 − |

📌 **说明**

* 这是**速度命令（reference）**，不是位置
* 是 **累加式**（按多次会越来越快）
* 建议评估时：
  👉 每次只按 **1–3 次**（0.1~0.3）

---

### 🔄 转向命令（整体旋转）

| 键   | 作用 | 改变的变量                     | 含义         |
| --- | -- | ------------------------- | ---------- |
| `q` | 左转 | `env.turn_rate_poi += 5°` | 载体期望逆时针角速度 |
| `e` | 右转 | `env.turn_rate_poi -= 5°` | 载体期望顺时针角速度 |

📌 **说明**

* 单位是 **角速度**（不是角度）
* 会被积累进 orientation tracking
* 不建议频繁乱按（否则 orientation error 会变大）

---

### 📏 高度控制（载体上下）

| 键   | 作用 | 改变的变量                     |
| --- | -- | ------------------------- |
| `1` | 降低 | `env.height_base -= 0.05` |
| `2` | 抬高 | `env.height_base += 0.05` |

📌 **用途**

* 测试对高度命令的响应
* 过大可能导致不稳（慎用）
